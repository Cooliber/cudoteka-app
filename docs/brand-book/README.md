# Księga Znaku Cudoteka

Kompleksowa dokumentacja identyfikacji wizualnej marki Cudoteka.

## Zawartość

Otwórz `index.html` w przeglądarce, aby zobaczyć pełną księgę znaku.

### Sekcje

1. **Tożsamość** — misja, wizja, wartości, atrybuty marki
2. **Kolory** — paleta podstawowa i akcentowe, zestawienia gradientów
3. **Typografia** — system Fraunces + Nunito Sans, skala typograficzna
4. **Logo** — warianty, strefa ochronna, zasady stosowania
5. **Elementy designu** — przyciski, karty, eyebrow, odstępy, cienie, animacje
6. **Głos marki** — ton, styl komunikacji, tagline, przykłady
7. **Zdjęcia** — styl fotografii, obróbka, ikony i emoji
8. **Zastosowania** —Where to apply the identity
9. **Specyfikacja** — wartości techniczne (CSS, breakpoints, easing)

## Pliki
